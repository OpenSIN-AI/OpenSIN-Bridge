OpenSIN-Bridge is a thin-client Chrome MV3 extension plus a server-side Cloudflare Worker backend. The extension handles browser-facing automation surfaces (background service worker, content scripts, popup UI), while the server hosts the private SaaS logic. Top-level structure: extension/, server/, docs/, scripts/, ops/, supabase/. Root package.json scripts include ext:dev, ext:package, server:dev, deploy:server, build, and a placeholder test script.
